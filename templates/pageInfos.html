Données de Retour possibles :

Code Status Message Explication
0 OK <Object> La requête a bien été effectuée, le JSON retourné est l'object créé dans la BDD
1 ERROR <Key> key is missing Une des clés necessaires à la création d'un objet est manquante (<Key> sera remplacé par le nom de la clé pour plus d'info)
2 ERROR The SQL Query cannot be performed Une des informations rentrées n'est pas bonne
par exemple: l'ID rentré pointe sur un objet inexistant
3 ERROR The target has too many container on it already ! Assez explicite: Le bateau cible a déjà trop de conteneurs dessus
4 ERROR Need more space to add this product Les produits demandés à l'ajout prennent trop d'espace pour le conteneur cible
